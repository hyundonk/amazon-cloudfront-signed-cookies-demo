How it works:

  1. User submits login credentials to Node.js server
  2. Server validates credentials and generates signed cookies
  3. Server sends Set-Cookie headers with CloudFront signed cookies
  4. Browser automatically includes cookies in subsequent requests
  5. CloudFront validates cookies for restricted content access
